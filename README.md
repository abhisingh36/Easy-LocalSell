# 🛒 Easy - Local Buy & Sell Marketplace

A full-stack local marketplace built with the **MERN Stack**, allowing users to buy and sell second-hand items with real-time chat.

## 🚀 Features
- 🔐 JWT Authentication
- 📦 CRUD Item Listings
- 💬 Real-time Chat (Socket.io)
- 📍 Location-based Listings
- 🔍 Search & Filters
- ⭐ Reviews & Ratings
- 📱 Responsive UI

## 🛠️ Tech Stack
**Frontend:** React, Vite, Tailwind CSS, React Router, Socket.io Client

**Backend:** Node.js, Express.js, MongoDB, Mongoose, Socket.io, JWT, bcryptjs

## ⚙️ Installation

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
npm run install:all
npm run dev
```

Create `.env` files:

**Backend**
```env
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

**Frontend**
```env
VITE_API_BASE=http://localhost:5000/api
```
